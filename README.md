# RealTimeChatApp

A real-time chat application built using React, Firebase, and Vite. It features real-time messaging, user authentication, and file storage for images and audio. The application is designed to be responsive and user-friendly.

## Features
- **Real-time Messaging**: Instant messaging between users.
- **User Authentication**: Secure login and user management.
- **File Storage**: Support for uploading and sharing images and audio files.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Getting Started
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/dsasank04/RealTimeChatApp.git
   cd RealTimeChatApp
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Application**:
   ```bash
   npm run dev
   ```

## Technologies Used
- **React**: Front-end library for building the user interface.
- **Firebase**: Backend-as-a-Service (BaaS) for real-time data synchronization and user authentication.
- **Vite**: Fast build tool for front-end development.
